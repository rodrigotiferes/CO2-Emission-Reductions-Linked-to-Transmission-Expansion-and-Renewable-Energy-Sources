Dear user,

This repository contains a MATLAB script that generates graphical results from a case study in the SDDP showing that new transmission facilities in Brazil, which will contribute to the connection of +876 MW of renewable generation (+615 MW of solar photovoltaic generation and +261 MW of biomass-based thermal generation), also have a direct impact on reducing emissions. 

The research paper, which contains these results, is titled "A Future-Looking Methodology for Estimating CO2 Emissions Reductions Linked to Transmission Projects That Increase Renewable Generation Capacity" and features in the IEEE Latin America Transactions (Submission ID: 10236).

To generate the results, please run it in your MATLAB (or Octave) command window. The script will automatically generate the results corresponding to figures 6, 7, 8, 9, 10, and 11. There is no need for any additional script or data. This script already contains all the results regarding CO2 emissions and reductions and emissions. 

Please be informed that the SDDP data itself is very large, but it can be publicly accessed at <https://www.gov.br/mme/pt-br/assuntos/secretarias/sntep/publicacoes/plano-decenal-de-expansao-de-energia/pde-2031/estudos-de-apoio>.
